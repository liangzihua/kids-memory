## 一、截图操作步骤

1. 用 **Chrome** 打开 `icon-512.html`
2. 按 F12 → 点"Elements"面板 → 找到 `<div class="icon">` → 右键 → **Capture node screenshot**
3. 保存为 `icon-512.png`

4. 用 Chrome 打开 `screenshots.html`
5. 同样方式，分别右键三个 `<div class="phone s1/s2/s3">` → Capture node screenshot
6. 保存为 `screenshot-1.png`、`screenshot-2.png`、`screenshot-3.png`

---

## 二、隐私政策外链部署（免费）

华为要求提供一个**可公开访问的隐私政策 URL**，最简单的方式是用 GitHub Pages：

1. 在 GitHub 新建仓库，名称如 `kids-memory-privacy`
2. 把 `privacy.html` 重命名为 `index.html` 上传
3. 仓库设置 → Pages → 选 main 分支 → Save
4. 几分钟后得到地址：`https://你的用户名.github.io/kids-memory-privacy/`
5. 这个地址填入华为应用市场"隐私政策 URL"字段
